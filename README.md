# 文本转换工具

一个简单实用的在线文本转换工具，可以将多行文本转换为不同格式的分隔文本。

## 功能特点

- 支持逗号(,)和分号(;)分隔
- 支持单引号(')和双引号(")包围
- 分隔符可自由组合
- 实时预览转换效果
- 一键复制结果

## 本地开发

1. 克隆项目
```bash
git clone [your-repo-url]
cd text-converter
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行项目
```bash
python main.py
```

4. 访问
打开浏览器访问 http://localhost:8001

## 技术栈

- FastAPI
